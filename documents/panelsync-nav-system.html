Issue 01 — Tone notes
Kyle should never seem exasperated — only gently weary. He chose this job.
Major Patriot doesn't understand irony yet. That's the arc. Don't let him be self-aware before Issue 03.
A: Tapping a world tool with no existing content open → full swap. Sidebar tree updates to show that tool's items. Back button or re-tapping Script returns you.
B: Tapping a world tool while writing → contextual prompt appears: "Open alongside?" or "Replace view?" If alongside: script compresses, panel opens right. Dismisses with ✕.
C: Notes from focus mode → always opens as a sheet overlay. Doesn't interrupt writing state. Swipe down or Done to dismiss. Quick-capture shortcut triggers same sheet.
